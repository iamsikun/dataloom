# Paper Context

        ## Title
        Optimal Real-Data Allocation for Synthetic-Data-Augmented Inference

        ## Topic
        Large language models make synthetic data inexpensive but biased, so researchers must split scarce real data between direct estimation and calibrating the generator. The paper studies this allocation problem, derives the marginal-value condition for optimal calibration, and shows that positive fixed-share baselines over-calibrate when the synthetic budget grows with real data and generator bias declines quickly.

        ## Contributions
        1. Exact corrected first-order condition (v_n + c x^{-2beta})^2 = 2 beta a c x^{-(2beta+1)} for the optimal calibration size, which keeps the synthetic-variance term omitted by fixed-share baselines.
2. Five-regime allocation phase diagram in (beta, rho) space; in the AI-relevant regime (rho approx 1, beta>1/2) the optimal calibration size grows as n^{2/(2beta+1)} so the calibration share vanishes.
3. Adaptive grid-minimization estimator with safety check x B_n(x) < a that falls back to all-real, plus an oracle inequality under uniform relative consistency of the pilot risk.
4. Separation of MSE-optimal point estimation from valid inference: naive Wald around the MSE-optimal estimator can undercover, while validation-debiased intervals restore coverage at a constant variance cost.
